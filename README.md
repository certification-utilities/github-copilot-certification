# GitHub Copilot Certification

Este repositório contém recursos para auxiliar na preparação para a certificação do GitHub Copilot.

## Como usar

Para obter um resumo completo da prova de certificação do GitHub Copilot, execute o prompt customizado disponível neste repositório, seguindo os passos a seguir:

1. Acesse o arquivo `.vscode/mcp.json` deste repositório e inicialize todos os servidores MCP clicando em Start:

![start mcp](/docs/img/mcp.png)

2. Abra o GitHub Copilot Chat no VS Code

3. Digite `/github-copilot-certification` para invocar o prompt

![call prompt](/docs/img/prompt.png)

4. O Copilot irá gerar um resumo detalhado sobre a certificação buscando das fontes oficiais e guiará para as proximas etapas de estudo.

![return prompt](/docs/img/study-guide.png)
